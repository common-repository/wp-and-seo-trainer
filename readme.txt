WP And SEO Trainer provides tutorials and lessons for using WordPress CMS as well as training videos for search engine optimization best practices.

== Description ==

WP And SEO Trainer is a simple and lightweight plugin that provides easy to understand step by step tutorials for using WordPress CMS as well as Search Engine Optimization Tips and Suggestions through video tutorials right inside your Dashboard.

= Features =

* Plugin automatically loads the relevant lesson depending upon the screen you are viewing.
* Mobile friendly and fully responsive videos (also available in full screen HD viewing).
* Easy to follow step by step instructions for visual learners.
* Video lessons for popular plugins such as YOAST SEO Plugin, Contact Form 7 Plugin and All in One SEO Pack Plugin.
* Tips and best practices for Akismet Plugin, Limit Login Attempts Plugin and various sections of WP Screen options.
* It explains SEO best practices without the techno-jargon talk, and, according to Google Webmaster Guidelines.
* Non-Intrusive Show/Hide button to reveal the lesson only when you want to learn more.

= How to Implement Suggestions found in WP And SEO Trainer =
WP And SEO Trainer plugin by will not implement any SEO techniques by itself. You need to do search engine optimization yourself. However because each step is detailed thoroughly, we believe you can learn how to use WordPress CMS and implement the suggestions easily without leaving your Dashboard.


== Installation ==

= Manual Installation =

1. Upload the `wp-and-seo-trainer` folder to the `/wp-content/plugins/` directory
2. Activate the WP And SEO Trainer plugin through the 'Plugins' menu in WordPress
3. You will now see WP And SEO Trainer button, press on it to reveal the lessons right inside the WP dashboard

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't even need to leave your web browser. To do an automatic install, log in to your WordPress admin panel, navigate to the Plugins menu and click Add New.

In the search field type "WP And SEO Trainer" and click Search Plugins. Once you've found the plugin you can view details about it such as the description. Most importantly, you can install it by clicking _Install Now_ and _Activate_.
